# GoalOS AI

**Turn screen time into goal time.**

An open-source AI productivity personality OS that helps you align mobile usage with your goals.

## Projects

| App | Stack | Description |
|-----|-------|-------------|
| [**goalos-web**](./goalos-web) | Next.js, React, Tailwind | Web demo with premium UI — local-first, no backend required |
| [**goalos-android**](./goalos-android) | Kotlin, Jetpack Compose | Native Android app with UsageStatsManager integration |

## Quick start (Web)

```bash
cd goalos-web
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Core product loop

1. **Goal setup** — Define what you're working toward
2. **Productivity DNA** — Discover your patterns and identity
3. **Alignment score** — Daily score based on behavior vs goal
4. **AI Coach** — Context-aware coaching and next actions
5. **Intent Gate** — Pause before distracting apps
6. **Focus Sprint** — Timed deep-work blocks
7. **Weekly identity** — Track who you're becoming
